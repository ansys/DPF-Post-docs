Installation#

Install using pip#

pip is the package installer for Python. To use PyDPF-Post with Ansys 2021 R1 or later, install the latest version with:

pip install ansys-dpf-post

Install using a wheel file#

If you are unable to install PyDPF-Post on the host machine due to network isolation, download the latest wheel file from PyDPF-Post GitHub or PyDPF-Post PyPi.

Install in development mode#

If you want to edit and potentially contribute to DPF-Post, clone the repository and install it using pip with the -e development flag:

git clone https://github.com/pyansys/pydpf-post
cd pydpf-post
pip install -e .
