Solution object¶

The Solution object is the entry point for browsing the contents of the results file. Use the following code to load the file and obtain the instance.

>>> from ansys.dpf import post
>>> from ansys.dpf.post import examples
>>> solution = post.load_solution(examples.multishells_rst)
class ansys.dpf.post.dpf_solution.DpfSolution(data_sources, model)¶

Main class of post result API.

get_result_info()¶

Returns information about the result file.

Examples

>>> from ansys.dpf import post
>>> result = post.result("file.rst")
>>> print(result.get_result_info())
