On this page

Module containing the DpfSolution class and its children. Each class highlights an analysis type, and provides hard-coded methods to get a result object regarding the wanted result type.

Module containing also the DpfComplexSolution class, child of DpfSolution class. In addition to the classic APIs, the complex result introduces an amplitude evaluation.
