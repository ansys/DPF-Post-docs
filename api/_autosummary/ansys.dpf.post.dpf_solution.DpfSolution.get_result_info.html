get_result_info#

DpfSolution.get_result_info()#

Get result file information.

Examples

>>> from ansys.dpf import post
>>> from ansys.dpf.post import examples
>>> solution = post.load_solution(examples.static_rst)
>>> print(solution.get_result_info()) 
Static analysis

Physics Type: ...
